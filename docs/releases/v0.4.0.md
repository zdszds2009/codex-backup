# Release v0.4.0 — Auto demo with real data output

## What's new

### Auto demo with real data output

A new `demo/auto_demo.py` script that runs a complete export-inspect-restore
cycle using actual Codex data. No simulation, no fake data.

### New features

- **auto_demo.py** — Run `python demo\auto_demo.py` to see a full 5-step demo
  with your own Codex project data
- **CODEX_HOME_OVERRIDE** — `build_restore_package.py` now supports the
  `CODEX_HOME_OVERRIDE` environment variable for custom Codex home paths

### Demo assets
