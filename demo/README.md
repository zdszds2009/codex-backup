# Demo

This folder contains a static demo landing page for `Codex Backup`.

## Files

- `index.html`: a one-page visual demo of the export/restore flow

## How to use it

Open `index.html` in a browser and take a screenshot, or host it on GitHub Pages.

## Suggested release use

- Screenshot the hero section for the GitHub release
- Link the page from the README
- Use it as a demo asset in the `Codex for Open Source` application
